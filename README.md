生命游戏，一个细胞自动机的示例，设置一个初始状态，观察它随着时间演化的过程。

### 规则

一个无限的二维正交方格网络，每一个方格被称为一个细胞，每个细胞只有两个状态 `死亡` 或者 `存活`
每个细胞会和它周围的八个邻居发生相互作用

- 任何存活的细胞，如果周围少于两个存活的邻居，则死亡
- 任何存活的细胞，周围存在多余三个邻居，则会死亡
- 任何存活的细胞，如果周围有两个或三个存活邻居，则会继续存活到下一代
- 任何存活的细胞，如果周围正好有三个存活邻居，则会变为存活细胞
